## Description
REST-service for a simple chat with JWT-authentication

### Tech stack
* Java 17
* Maven
* Spring Boot
  * Web
  * Data Jpa
  * Validation
  * Security
* PostgresSQL
* JSON Web Tokens (JWT)

## Build
    $ git clone https://github.com/ksilisk/chat-rest-service.git
    $ cd chat-rest-service
    $ ./mvnw install
## Run
    $ ./mvnw spring-boot:run
## Tests
    $ ./mvnw test
